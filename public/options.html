Settings

Your files get written straight to your PC. Pick a folder, then tune how Stuff Grabber looks and downloads.

Save location
Example: C:\Users\You\Videos\HLS. Stored in this browser. The folder is created if it is missing.
On-page controls
When off, only the toolbar popup lists streams. Queue and badges still work either way. Toggles apply to open tabs automatically.
Off by default. When on: hover an image for the download popover, Images list in the popup and floating panel, and right‑click save-as. Applies to open tabs automatically.
Recording
On by default. While recording, the video goes fullscreen on top with controls so you get the cleanest capture. Turn this off if you’d rather leave the page looking normal.
Download quality
Applies when ffmpeg re-encodes HLS/DASH to MP4. You can switch mid-download.
Asking only applies to YouTube-style and other social detections that use yt-dlp.
Limits merged video resolution. Leave blank for the built-in default. Enter 0 for no height limit.
Appearance
Match my browser follows light or dark. Match the page samples that tab’s theme color, CSS tokens, and UI accents so the grabber blends in.
Used for Always light, Always dark, and Match my browser. Hidden when matching the page.

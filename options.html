Where to save video

Files are written by ffmpeg on your PC. Paste a full folder path. Four downloads can run at once; extras wait in line.

Example: C:\Users\You\Videos\HLS. It is stored in this browser so you do not have to re-enter it; change it here anytime. We create the folder if it is not there.
When off, only the toolbar popup lists streams. Same download queue and badges either way.
yt-dlp downloads need the real page URL; the extension already prefers that. Asking only applies to YouTube-style and other social detections that use yt-dlp.
Limits merged video resolution (e.g. 1080 for 1080p). Leave blank for the built-in default. Enter 0 for no height limit.
